# Themes

This site uses CSS variables for colors and spacing. Edit `static/index.css` and adjust values under `:root` and the dark section.

## Variables

```css
:root {
  --bg: #ffffff;
  --fg: #0b0f19;
  --accent: #3b82f6;
}
```

> Tip: Prefer accessible contrast ratios (AA+).
